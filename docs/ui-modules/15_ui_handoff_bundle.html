Reusable UX Patterns

Searchable Picker

Use async search for customers, suppliers, products, owner accounts, roles, categories, brands, UOM, and tax groups.

Document Workspace

Quote, order, invoice, PO, receipt, and return screens should use summary header, item table, totals, links, PDF actions, and activity.

Allocation Drawer

Keep payment summary, open documents, editable allocation values, and remaining balance visible together.

Guided Stepper

Best for onboarding, product creation, invoice creation, purchase receipt creation, and service agreement setup.

Desktop And Mobile Rules

Desktop

  • prefer table-first layouts for operational modules
  • keep sticky filters and sticky totals where flows are item-heavy
  • use tabs for deep entity workspaces
  • use drawers for quick context rather than route churn

Mobile

  • prioritize lookup, capture, summary, and approval flows
  • replace dense tables with expandable cards
  • use full-screen pickers for async search
  • keep primary action pinned at bottom in steppers

Implementation Checklist
